40 - vue全解(起手式1)

1.vue 简介

  • JavaScript 框架
  • 简化 Dom 操作
  • 响应式数据驱动
  • 开发环境版本(类似完整版的 jQuery)，生产环境版本(类似 mini 版本的 jQuery)
  • 可以根据官方步骤来创建一个 vue 项目，也可以使用@vue/cli 来搭建
    1.导入开发版本的vue.js
    2.创建vue实例对象，设置el属性和data属性
    3.使用简洁的模板语法把数据渲染到页面上
    <div>
        {{message}}
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
        let vue = new Vue({
            el: "#app",
            data: {
                message: "hello vue!!"
            }
        });
    </script>

2.el 挂载点

  • vue 实例的作用范围是什么呢？
    答：Vue 会管理 el 选项命中的元素以及内部的后代元素
  • 示例
    <div id = "app">
        {{message}}
        <span>{{message}}</span>
    <div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
        let vm = new Vue({
            el: "#app",
            data: {
                message: "欢迎使用vue"
            }
        });
    </script>

  • 使用可以使用其它的选择器?
    答：可以使用其它的选择器，但是建议使用 ID 选择器

  • 是否可以设置其它的 dom 元素呢? 答：可以使用其它的双标签，不能使用 HTML 和 BODY

3.data 数据对象(支持对象和函数，组件必须要使用函数)

  • Vue 中用到的数据定义在 data 中
  • data 中可以写复杂类型的数据
  • 渲染复杂类型数据时，遵守 js 的语法即可
  • 组件的定义只接受function
  • 示例
    <div id = "app">
        <h1>{{message}}</h1>
        <h2>{{school.name}}{{school.mobile}}</h2>
        <ul>
            <li>{{campus[0]}}</li>
            <li>{{campus[1]}}</li>
            <li>{{campus[2]}}</li>
        </ul>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <script>
        let vm = new Vue({
            el: "#app",
            data: {
                message: "大家好！！",
                school: {name: '周瑞发', mobile: '1301121241'},
                campus: ["北京","上海","广州","深圳"]
            }
        });
    </script>

4.Vue.component 组件完整版才支持

  • 定义一个名为button-counter，名字可更改的新组件
  • component 第一个参数为你定义组件的名字，第二个参数跟 Vue 实例 options 一样
  • 组件可以无限复用，data 必须是一个函数，避免引用同样的内存地址，导致数据错乱
  • 建议大写组件名，以防和 html 标签发生冲突
  • 示例
    Vue.component("Button-counter",{
        data(){
            return{
                count: 0
            }
        },
        template: `
            <button @click="count++">你点击了我{{count}}下</button>
        `
    });

    new Vue({
        el: "#app",
        template: `
            <Button-counter></Button-counter>
        `
    });

或

    import Demo from "./Demo.vue";
    new Vue({
        el: "#app",
        components: {Demo},
        template: `
            <div>
                <Demo></Demo>
            </div>
        `
    });

5.什么是 vue 的生命周期

答：Vue 实例有一个完整的生命周期，也就是从开始创建、初始化数据、编译模板、挂载 Dom、渲染->更新->渲染、卸载等一系列过程，这就是 Vue 的生命周期，通俗说这就是 Vue 实例从创建到销毁过程，这就是生命周期。

6.生命周期四个钩子

  • created
  • 类型： Function
  • 详细：在实例创建完成后立即被调用
  • 示例
    new Vue({
        created(){
            console.log("Vue实例创建完成后立即就会执行该函数");
        }
    })
  • mounted
  • 类型：Function
  • 详细：已经挂载了选择器，就会触发该函数，这时候 el 就已经被$el 替换了
  • 示例
    new Vue({
        el: "#app",
        mounted(){
            console.log("已经挂载了el，就会触发该函数");
        }
    })
  • updated
  • 类型：Function
  • 详细：同组件下 data 对象数据发生了改变就会触发该函数
  • 示例
    new Vue({
        el: "#app",
        data(){
            return {
                n: 0
            }
        },
        methods: {
            add1(){
                this.n += 1;
            }
        },
        template: `
            <div>
                {{n}}
                <button @click="add1">+1</button>
            </div>
        `,
        updated(){
            console.log("数据更新或者页面更新会执行该函数");
        }
    })
  • destroyed
  • 类型：Function
  • 详细：组件被销毁后调用该函数，同组件才会触发该函数
  • 示例
    import Demo from "./Demo.vue"
    new Vue({
        el: "#app",
        components: {Demo}
        data(){
            return{
                isShow: true
            }
        },
        template: `
            <button @click="isShow=false">隐藏</button>
            <Demo v-if="isShow"></Demo>
        `
    })

7.使用 props 传递数据

  • 详细：组件不仅仅是要把模板的内容进行复用，更重要的是组件之间的通信，通常父组件的模板中包含子组件，父组件要向子组件传递数据或者参数，子组件接收到后根据参数的不同来进行对应的渲染。数据传递在 vue 组件可以通过 props 来实现
  • 示例
    // child.vue
    <template>
      <div>{{message}}</div>
    </template>

    <script>
      export default {
        props: ["message"]
      };
    </script>

    <style></style>


    // father.vue
    <template>
      <div>
        <input type="text" v-model="message" />
        <child :message="message"></child>
      </div>
    </template>

    <script>
    import child from "./child.vue";
    export default {
      data() {
        return {
          message: ""
        };
      },
      components: { child }
    };
    </script>

    // main.js
    import father from "./father.vue";
    new Vue({
      el: "#app",
      components: { father },
      template: `
        <div>
          <father></father>
        </div>
      `,
    });
