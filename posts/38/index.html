38 - webpack

1.安装 webpack–不建议安装在全局

  • npm info webpack 可以打印出 webpack 的所有信息
  • npm install webpack -g在全局安装 webpack
  • npm uninstall -g webpack在全局卸载 webpack

2.webpack 是干嘛用的？

  • 转译代码(Es6 转 Es5)
  • 构建 build
  • 代码压缩
  • 代码分析

3.用 webpack 生成 js

  • mkdir webpack-demo(文件名可修改)
  • cd webpack-demo
  • npm init -y初始化一下目录
  • npm install webpack webpack-cli –save-dev(安装依赖，建议在局部安装，这里报错你 package.json 重名了)
  • 创建一个 src 目录，目录下创建一个 index.js
  • ./node-modules/.bin/webpack 或者 npx webpack 运行本地目录 webpack
  • 如何消除警告(WARNING in configuration…)
    1.在目录加上webpack.config.js文件(添加一下代码)
    module.exports = {
        mode: "development" // development开发模式，production用户模式
    }
  • webpack 配置 entry 和 output
    module.exports = {
        mode: "development",
        entry: "./src/index.js", // 转义代码路径
        output: {
            filename: "index.js" // 转义完成后dist目录下的name
        }
    }

4.理解 webpack 文件名中的 hash 的用途(http 缓存)

  • HTTP 响应头中的 Cache-Control
  • 如果你访问一个网址，访问的 html 会引入 1.css，2.css，1.js…，如果你第二次在访问的时候就会有部分文件会缓存在你的硬盘中，在你缓存有效时间内，在进行访问这个网站，就不需要在次请求服务器，直接从你缓存硬盘中获取即可，能达到超快访问
    module.exports = {
        mode: "development",
        entry: "./src/index.js",
        output: {
            filename: '[name].[contenthash].js'
        }
    }
    1.每次运行webpack的时候，都会生成新的hash文件，如何解决？
    答：每次运行webpack，把dist目录删掉，在生成即可
    解决：在package.json中scripts里面添加一句 "build" : "rm -rf dist && webpack"

    2.在命令行运行yarn build 即可

5.用 webpack 生成 HTML

  • yarn add html-webpack-plugin –dev
  • 需要在 src 目录下创建 assets 目录，在 assets 目录创建 html 文件
  • 在 webpack.config.js 添加如下代码
    const HtmlWebpackPlugin = require("html-webpack-plugin");
    module.exports = {
        plugins : [new HtmlWebpackPlugin({
            title: '<% htmlWebpackPlugin.options.title %>',  // 如果想用html的title,需要在这里添加 <% htmlWebpackPlugin.options.title %>
            template: 'src/assets/index.html' // 需要在src创建assets目录
        })]
    }

6.webpack 引入 css(import 导入)

  • yarn add css-loader –dev
  • yarn add style-loader –dev
  • 注意：上面下载错误切换一下源下载即可，yarn config set registry https://registry.npmjs.org/
  • 在 webpack.config.js 添加如下代码
    module.exports = {
        module: {
            rules: [
                {
                    test: /\.css$/i,
                    use: ["style-loader","css-loader"]
                }
            ]
        }
    }

7.使用 webpack-dev-server

  • yarn add webpack-dev-server –dev
  • 在 webpack.config.js 添加如下代码
    module.exports = {
        devtool: "inline-source-map",
        devServer: {
            contentBase: './dist'
        }
    }
  • 在 package.json 里面 scripts 属性添加如下代码
    "start": "webpack-dev-server"
  • 运行 yarn start

8.如何生成 css 文件

  • yarn add mini-css-extract-plugin –dev
  • 在 webpack.config.js 添加如下代码
    const MiniCssExtractPlugin = require("mini-css-extract-plugin");
    module.exports = {
        plugins: [
            new MiniCssExtractPlugin({
                filename: '[name].css',
                chunkFilename: '[id].css'
            })
        ],
        module: {
            rules: [
                {
                    test: /\.css$/,
                    use: [
                        {
                            loader: MiniCssExtractPlugin.loader,
                            options: {
                                publicPath: '../',
                                hmr: process.env.NODE_ENV === 'development'
                            },
                        },
                        'css-loader',
                    ]
                }
            ]
        }
    }

9.使用两个 webpack.config.js

  • 切换两种模式(用户模式 | 开发模式)
    1.需要拷贝多一份webpack.config.js,如果是用户模式则把css生成文件,开发模式怎用import导入
    2.在webpack.json下scripts属性
        "build" : "rm -rf dist && webpack --config webpack.config.prod.js",
        "start" : "webpack-dev-server"

10.用 webpack 引入 SCSS

  • yarn add sass-loader dart-sass webpack –save-dev
  • 在 webpack.config.js 添加如下代码
module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/i,
        use: [
          'style-loader',
          'css-loader',
          {
            loader: "sass-loader",
            options: {
            implementation: require('dart-sass')
            }
          }
        ],
      },
    ],
  },
};

11.用 webpack 引入 less

  • yarn add less-loader –dev
  • 需要安装一下 less 环境 yarn add less –dev
  • 在 webpack.config.js 添加如下代码
module.exports = {
  module: {
    rules: [
      {
        test: /\.less$/,
        loader: ["style-loader","css-loader","less-loader"]
      },
    ],
  },
};

12.用 webpack 引入图片

  • 会自动帮你添加上哈希值
  • yarn add file-loader –dev
  • 在 webpack.config.js 添加如下代码
    module.exports = {
    module: {
        rules: [
            {
                test: /\.(png|svg|jpg|gif)$/,
                use: ['file-loader'],
            }
        ]
    }
}
