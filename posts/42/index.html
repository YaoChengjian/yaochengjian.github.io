42 - vue全解(模板、指令与修饰符)

1.模板 template 三种写法

  • Vue 完整版，写在 HTML 里
  • 示例
    <div id = "app">  // => HTML
        {{n}}
    <div>
  • Vue 完整版，写在选项里
  • 示例
    <div id = "app">  // => HTML
    <div>

    new Vue({
        el: "#app",
        data(){
            return { n:0 }
        },
        template: `
            <div>
                {{n}}
            </div>
        `
    })
    // 注意一个细节：div#app会被template替代
  • Vue 运行时版，配合 vue 单组件文件
  • 示例
    // vue单组件文件
    <template>
        <div>{{n}}</div>
    <template>

    <script>
        export default{
            data(){
                return { n:0 }
            }
        }
    </script>

    // 另外一个地方引入该vue文件
    import Demo import "Demo.vue";
    new Vue({
        el: "#app",
        render(h){
            return h(Demo)
        }
    })

2.指令

  • v-text
  • 预期：string
  • 详细：更新元素的 textCont。如果要更新部分的 textContent，需要使用{{meg}}来差值
  • 示例
    <div id = "app">
        <span v-text="msg"></span>
        <!-- 和下面一样 -->
        <span>{{msg}}</span>
    </div>

    <script>
        new Vue({
            el: "#app",
            data(){
                return{
                    msg: '你好呀！'
                }
            }
        })
    </script>
  • v-html
  • 预期：string
  • 详细：更新元素 innerHTML。可以识别 html 标签，并且渲染在页面上
  • 示例
    <div id = "#app">
        <p v-html="msg"></p>
    <div>

    <script>
        new Vue({
            el: "#app",
            data(){
                return {
                    msg: "<strong>你好呀！</strong>"
                }
            }
        })
    </script>
  • v-pre
  • 不需要表达式
  • 详细：不需要将 {{}} 进行编译，会把{{}}也显示出来
  • 示例
    <div id = "app">
        <span v-pre>{{n}}</span>
    </div>

    <script>
        new Vue({
            el: "#app",
            data(){
                return{
                    n: 0
                }
            }
        })
    </script>
  • v-bind
  • 缩写：:
  • 预期：任何(变量、字符串…)
  • 详细：在绑定 class 或 style 属性时，支持其它类型的值，如数组或对象
  • 示例
    <!-- 绑定一个属性 -->
    <img v-bind:src="imageSrc">

    <!-- 缩写 -->
    <img :src="imageSrc">

    <!-- class绑定 -->
    <div :class="{red:isRed}"></div>
    <div :class="[classA, classB]"></div>

    <!-- style 绑定 -->
    <div :style="{color:'red'}">111</div>
    <div :style="[styleObjectA, styleObjectB]"></div>
  • v-on:事件名
  • 缩写：@事件名
  • 详细：绑定事件
  • 示例
    <button v-on:click="add1"> +1 </button>

    <!-- vue做了处理，不会马上执行xxx，还是会等你点击执行 -->
    <button @click="xxx(1)">xxx</button>

    <!-- 还能放其它可执行代码 -->
    <button @click="n+=1">{{n}}</button>
  • v-if && v-else
  • 详细：
    1.根据表达式真假切换元素的显示状态，本质是通过 dom 来切换显示状态。true 为显示状态，false 为隐藏状态。
    2.v-else 必须要和 v-if 匹配使用，v-else 不能单独使用，只有 v-if 的值为 false 时，v-else 模块才会显示出来
  • 示例
    <template>
      <div>
        <button @click="show">{{message}}</button>
        <div class="red" v-if="isShow"></div>
        <div class="blue" v-else></div>
      </div>
    </template>

    <script>
    export default {
      data() {
        return {
          message: "隐藏",
          isShow: true
        };
      },
      methods: {
        show() {
          this.isShow = !this.isShow;
          this.message = this.isShow ? "隐藏" : "显示";
        }
      }
    };
    </script>

    <style Lang="css" spoced>
    .red,.blue{
        width: 200px;
        height:200px;
    }
    .red{
        border: 1px solid red;
    }
    .blue{
        border: 1px solid blue;
    }
    </style>
  • v-show
  • 用法：根据表达式真假值，切换元素的 display CSS 属性
  • 示例
    <template>
        <div>
            <button @click="isShow=false">点击</button>
            <p v-show="isShow"></p>
        </div>
    </template>

    <script>
        export default{
            data(){
                return {
                    isShow:true
                }
            }
        }
    </script>
  • v-for
  • 详细：for(value,key) in 对象或数组
  • 注意：使用 v-for 就必须要在后面加上:key，否则会一直出现警告
  • 示例
    <template>
        <div>
            <ul>
                <li v-for="(value,key) in users" :key="key.id"></li>
            </ul>
        </div>
    </template>

    <script>
        export default {
            data(){
                return{
                    users: ["frank","tom","lucy","juck"]
                }
            }
        }
    </script>

    <style></style>

3.修饰符

1.v-on 支持的修饰符如下 ↓ ：

  • 事件修饰符
  • 详细：vue.js 提供了事件修饰符。修饰符是由.开头的指令来表示的
  • 示例
    @click.stop="add"  // 表示阻止事件传播/冒泡
    @click.prevent="add"  // 表示阻止默认动作
    @click.stop.prevent="add"  // 修饰符可以串联
    @click.once="doThis"  // 点击事件将只会触发一次
  • 按键修饰符
  • 详细：在监听键盘事件时候，我们经常需要检查详细的按键。Vue
  • 按键码：为了在必要的情况下支持旧浏览器，Vue 提供了绝大常用的按键码的别名：
    <input @keyup.13 = "submit"> // 按下enter键执行该函数

    为了在必要的情况下支持旧浏览器，Vue提供了绝大多数常用的按键码的别名
    .enter (回车)
    .tab (Tab)
    .delete (delete(删除)/BackSpace(退格))
    .space  (空格键)
    .up (上箭头)
    .down (下箭头)
    .left (左箭头)
    .right (右箭头)
  1. 理解 vue 中的.sync 修饰符
  • 详细：在使用 vue 的过程中，我们有时会遇到如下场景：子组件需要引用父组件的一个属性，并且修改。由于 vue 是不允许对外部的数据直接修改的，所以便需要在两个组件之间建立一个通信，让父组件去监听子组件发出的请求，然后再对相应的值做出改变。
  • 用法：
    1.我们首先需要在子组件上面，建立一个事件，然后写上事件名和变化的值
    2.vue内置了EventBus为$emit，可以用来触发事件
    this.$emit("update:money",money-100)

    1.然后在父组件上面监听这个事件的触发，将变化的值传回来
    2.同时vue将子组件改变的那个值，存进了$event这个内置变量中，这样父组件中就能获取到这个值
    v-on:update:money="total=$event"

    1.vue又将监听的部分写集成.sync修饰符，这样看起来更加的简洁，于是上面这一行代码也可以简写为如下形式：
    :money.sync="total"
  • 示例
Child.vue
    <template>
      <div class="green">
        <button @click="$emit('updata:money',money-100)">花钱</button>
        <span>{{money}}</span>
      </div>
    </template>

    <script>
    export default {
      props: ["money"]
    };
    </script>

    <style scoped>
    .green {
      border: 3px solid green;
    }
    </style>

Father.vue
    <template>
      <div class="red">
        App.vue 还有{{total}}
        <Child :money="total" v-on:update:money="fn"></Child>

        // 上面 v-on:update:money="fn" 等价于 :money.sync="fn"

        <Child :money.sync="fn"></Child>
      </div>
    </template>

    <script>
    import Child from "./Child.vue";
    export default {
      data() {
        return {
          total: 10000
        };
      },
      components: { Child },
      methods: {
        fn(event) {
          // event 是子组件传递过来的值，天生就存在
          this.total = event;
        }
      }
    };
    </script>

    <style scoped>
    .red {
      border: 3px solid red;
      padding: 30px;
    }
    </style>
