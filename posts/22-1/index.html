22-1 - [JS全解]Map 、 Set

1.Map([iterable]) 键值对为两个元素的数组例如:[1,‘one’]

1.map 的实例属性和操作方法
—-size : 返回成员总数
—-set(key,value) : 添加新的键值
—-has(key) : 是否有某个键
—-get(key) : 读取某个对应的值
—-delete(key) : 删除某个键
—-clear() : 清空

2.map 实例遍历的方法
—-keys() : 返回键名
—-values() : 返回键值
—-entires() : 返回键值对
—-forEach() : 遍历所有成员

利用map实现去重
    let array = [1,2,3,5,1,2,3,4]
    let map = new Map();
    for(let i = 0;i < array.length;i++>){
        if(map.has(array[i])){
            continue;
        }else{
            map.set(array[i],true);
        }
    }
    let newAry = [...map.keys()]

2.Set([iterable])对象允许你存储任何类型的唯一值(set 里面的值是唯一的)

方法和属性
—-size : 返回 set 对象的个数
—-add(value) : 在 set 对象尾部添加一个元素,返回该 set 对象
—-clear() : 移除 set 对象内的所有元素 —-delete(value) : 移除 set 中与这个值相等的元素
—-has(value) : 判断 set 中是否存在这个值 —-entires() : 返回键值对的数组(二维数组)
—-keys() : 返回键名
—-values() : 返回键值
—-entires() : 返回键值对
—-forEach() : 遍历所有成员
